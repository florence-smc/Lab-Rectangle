PROJECT TITLE: Rectangle
PURPOSE OF PROJECT: Calculate the area and perimeter of a rectangle
VERSION or DATE: May 5, 2017
HOW TO START THIS PROJECT: Right click on the Rectangle class and create a new instance of it
AUTHORS: Florence Shelley
USER INSTRUCTIONS: Right click on the Rectangle class and create a new instance of a Rectangle, 
then provide the width and height of your rectangle. Once this new instance has been created,
right click on your new instance and click on the getArea() method to calculate the rectangle's
area, getPerimeter() to calculate its perimeter, getWidth() to view the current width,
getHeight() to view the current height, and flip() to swap the values of the width and height
of the rectangle with one another.
